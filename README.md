# Проектно-технологическая практика

## Описание проекта

В рамках проектно-технологической практики выполнено моделирование деятельности гостиничного предприятия с использованием нотаций BPMN 2.0 и IDEF0, а также разработаны программные модули на языке Python.

Цель работы заключается в изучении методов анализа бизнес-процессов, освоении инструментов функционального и процессного моделирования, а также приобретении практических навыков программирования.

## Структура репозитория

### task1-analysis

Анализ деятельности гостиничного предприятия и выбор процесса для моделирования.

Содержимое:

- analysis.md — описание организации, участников и бизнес-процессов.

### task2-bpmn

Материалы по нотации BPMN 2.0 и модель процесса регистрации и размещения гостей.

Содержимое:

- notes_bpmn.md — конспект по BPMN 2.0;
- description.md — описание BPMN-модели;
- process_main.png — итоговая BPMN-диаграмма.

### task3-idef0

Материалы по нотации IDEF0 и функциональная модель гостиничного предприятия.

Содержимое:

- notes_idef0.md — конспект по IDEF0;
- description.md — описание функциональной модели;
- a0_context.png — контекстная диаграмма;
- a0_decomposition.png — диаграмма декомпозиции.

### task4-python

Программные модули на языке Python.

Содержимое:

- module1_basics.py — работа с переменными и арифметическими операциями;
- module2_conditions.py — использование условий и циклов;
- module3_functions.py — использование пользовательских функций;
- mini_project.py — мини-проект по расчету показателей деятельности гостиницы.

## Используемые технологии

- BPMN 2.0;
- IDEF0;
- Python;
- GitHub.

## Результаты работы

В ходе практики был выполнен анализ деятельности гостиничного предприятия, разработана BPMN-модель процесса регистрации и размещения гостей, построена функциональная модель IDEF0 и созданы программные модули на языке Python.
