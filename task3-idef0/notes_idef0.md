# Конспект по нотации IDEF0

## Назначение IDEF0

IDEF0 представляет собой методологию функционального моделирования, предназначенную для описания деятельности организации в виде взаимосвязанных функций. Нотация используется для анализа структуры процессов, определения входов и выходов функций, а также ресурсов и управляющих воздействий, необходимых для выполнения работ.

## Основные элементы IDEF0

### Функциональный блок

Функциональный блок отображается прямоугольником и представляет отдельную функцию системы.

Каждая функция отвечает на вопрос:

«Что выполняется в рамках рассматриваемой системы?»

### Входы (Input)

Входы располагаются с левой стороны блока.

Входом является информация или объект, который преобразуется в результате выполнения функции.

### Выходы (Output)

Выходы располагаются с правой стороны блока.

Выход представляет результат выполнения функции.

### Управление (Control)

Управляющие воздействия располагаются сверху блока.

Они определяют правила выполнения функции и влияют на процесс преобразования входов в выходы.

Примеры управления:

- стандарты обслуживания;
- тарифная политика;
- внутренние регламенты.

### Механизмы (Mechanism)

Механизмы располагаются снизу блока.

Они определяют ресурсы, обеспечивающие выполнение функции.

Примеры механизмов:

- персонал;
- оборудование;
- информационные системы;
- номерной фонд гостиницы.

## Уровни моделирования

IDEF0 предусматривает иерархическое представление системы.

Основными уровнями являются:

- контекстная диаграмма A-0;
- диаграмма декомпозиции A0;
- диаграммы последующих уровней детализации.

## Преимущества IDEF0

Использование IDEF0 позволяет:

- определить состав функций организации;
- установить взаимосвязи между функциями;
- выделить входы и выходы процессов;
- определить ресурсы и управляющие воздействия;
- представить деятельность организации в структурированном виде.
